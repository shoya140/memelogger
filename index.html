MEMELogger

Sensor data logging application for JINS MEME

MEMELogger(s) are sensor logging applications for JINS MEME ES (Developer version) and ES_R (Academic version). ■ Functions ・Visualizing sensor signals in real time. ・Recording senosr data as .csv file (even if the app is running in the background). ■ MEMELogger on iOS device ・Data will be stored under NSDocumentDirectory. ・Connect your iPhone to Mac/PC and find data by using iTunes File Sharing. ■ MEMELogger on Android ・Data will be stored at /sdcard/MEMELogger(-Academic)/ ・Find data by using Android File Transfer or ``$ adb pull ...``
